Instalação
--------

$ git clone https://github.com/brenddongontijo/api_ruby.git

$ rvm install 2.3.1

$ rvm use 2.3.1

$ gem install rails

$ bundle install

$ rake db:create && rake db:migrate

$ rails s


Rodar Suite de Testes
--------
$ rake db:test:prepare

$ bundle exec rspec
